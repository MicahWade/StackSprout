# 🌱 StackSprout

StackSprout is a modern, high-energy tech stack randomizer designed to help developers and teams break out of their comfort zones and discover new synergy in their development workflows.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone git@github.com:MicahWade/StackSprout.git
   ```
2. Navigate to the project directory:
   ```bash
   cd StackSprout
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## 🛠️ Commands

- `npm start` / `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run preview`: Previews the production build locally.
- `npm run type-check`: Runs TypeScript type checking.

## 🏗️ Tech Stack Options

The available technologies can be found and modified in `src/technologies.json`.
